# RedMango 網路點餐

Frontend使用React Hooks/Redux, Backend以.NET Core WebAPI, 實作登入/登出/註冊、餐點管理、購物車、訂單處理、分頁、關鍵字搜尋以及串接第三方支付等功能。

# 使用技術
前端 
* React with Redux
* TypeScript

後端 
* .Net Core 6 Web API
* Core Identity with JWT
* Automapper
* Azure Bob Storage
